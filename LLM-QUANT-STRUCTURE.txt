快速开始流程
═══════════════════════════════════════════════════════════

1️⃣  启动开发服务器
    ┌──────────────────────────────────────────────────┐
    │ $ cd comistrymo_blog                              │
    │ $ npm run dev                                     │
    │                                                   │
    │ 访问: http://localhost:4321/blog                  │
    └──────────────────────────────────────────────────┘

2️⃣  创建新文章
    ┌──────────────────────────────────────────────────┐
    │ $ cd src/content/blog/LLM/Quant/                  │
    │ $ cp template.mdx my-article.mdx                  │
    └──────────────────────────────────────────────────┘

3️⃣  编辑文章
    ┌──────────────────────────────────────────────────┐
    │ 修改 frontmatter（标题、描述、封面图等）          │
    │ 编写文章内容（Markdown/MDX）                      │
    │ 添加图片到 images/ 文件夹                         │
    └──────────────────────────────────────────────────┘

4️⃣  预览文章
    ┌──────────────────────────────────────────────────┐
    │ http://localhost:4321/blog/llm/quant/my-article   │
    └──────────────────────────────────────────────────┘

5️⃣  发布文章
    ┌──────────────────────────────────────────────────┐
    │ $ npm run build      # 构建                       │
    │ $ npm run preview    # 预览                       │
    │ $ git add .          # 提交                       │
    │ $ git commit -m "Add new blog post"               │
    │ $ git push           # 推送（自动部署）           │
    └──────────────────────────────────────────────────┘

图片管理方式
═══════════════════════════════════════════════════════════

方式一：本地相对路径（推荐）⭐
┌────────────────────────────────────────────────────────┐
│ 优势：自动优化、WebP转换、响应式、类型安全              │
│                                                          │
│ 位置：src/content/blog/LLM/Quant/images/                │
│                                                          │
│ 使用：                                                   │
│   frontmatter: image: "./images/cover.png"              │
│   内容中: ![图片](./images/figure1.png)                 │
│                                                          │
│ 推荐场景：所有场景（默认使用）                          │
└────────────────────────────────────────────────────────┘

方式二：Public 目录
┌────────────────────────────────────────────────────────┐
│ 优势：不需要构建、适合大量图片、可直接访问              │
│                                                          │
│ 位置：public/images/blog/LLM/Quant/                     │
│                                                          │
│ 使用：                                                   │
│   frontmatter: image: "/images/blog/LLM/Quant/cover.png"│
│   内容中: ![图片](/images/blog/LLM/Quant/figure1.png)  │
│                                                          │
│ 推荐场景：大量图片、频繁更新                            │
└────────────────────────────────────────────────────────┘
